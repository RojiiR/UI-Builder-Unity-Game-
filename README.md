# 🧩 Puzzle Builder (Unity)

Script **puzzlebuilder.cs** adalah tool untuk mempermudah pembuatan UI puzzle di Unity secara otomatis.

## ✨ Fitur
- Mendukung **7 jenis puzzle yang berbeda**
- Setup UI otomatis ke dalam **Hierarchy**
- Cukup **1 klik** untuk generate semua elemen
- Mempercepat workflow development

## 🚀 Cara Penggunaan

1. Import file `puzzlebuilder.cs` ke dalam project Unity kamu
2. Buka Unity Editor
3. Jalankan builder (biasanya melalui tombol di Inspector atau menu)
4. UI puzzle akan otomatis muncul di **Hierarchy**

## 🧠 Tentang Puzzle
Script ini menyediakan total **7 puzzle berbeda**, yang bisa kamu gunakan atau kembangkan lagi sesuai kebutuhan game kamu.

## 📂 Struktur
- `puzzlebuilder.cs` → Script utama untuk generate UI puzzle

## ⚙️ Requirements
- Unity (disarankan versi terbaru / kompatibel dengan project kamu)

## 📌 Catatan
- Pastikan scene kamu siap sebelum menjalankan builder
- Disarankan menggunakan project kosong atau scene khusus untuk testing
